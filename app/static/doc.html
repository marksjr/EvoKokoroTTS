Evo KokoroTTS

Local multilingual Text-to-Speech REST API using Kokoro-82M and Edge TTS. Convert text to audio with multiple voices, speed control, pitch adjustment, and real-time streaming.

ⓘ
Base URL: http://localhost:8880
The API also generates automatic interactive documentation at /docs (Swagger UI) and /redoc (ReDoc).

Unicode Normalization (NFC)

ⓘ
The API automatically normalizes all text to NFC (Unicode composed form). This ensures that characters like ã, ç, é, ô, ü are handled correctly, regardless of how they were typed or copied. Typographic quotes, em dashes, and Unicode ellipses are also converted.

How to use other languages

The synthesis language is determined by the voice. There is no separate language field in the POST to /tts or /tts/stream.

StepWhat to do
1Query GET /languages to see the languages supported in the current installation.
2Query GET /voices and choose a voice whose lang field matches the desired language.
3Send the text using that voice. The correct engine (Kokoro or Edge) will be chosen automatically.
ⓘ
Languages configured in this version: pt-br, en-us, en-gb, es-es, fr-fr, ja-jp, zh-cn, hi-in, it-it and de-de.
Voice examples: pf_dora (pt-BR), af_bella (en-US), bf_alice (en-GB), ef_dora (es-ES), ff_siwis (fr-FR), jf_alpha (ja-JP), zf_xiaoxiao (zh-CN), hf_alpha (hi-IN), if_sara (it-IT) and de_f_katja (de-DE). 
# English (Kokoro)
curl -X POST http://localhost:8880/tts \
  -H "Content-Type: application/json" \
  -d '{"text":"Hello. This is a test in English.","voice":"af_bella","speed":1.0,"format":"mp3"}' \
  --output english.mp3

# Spanish (Kokoro)
curl -X POST http://localhost:8880/tts \
  -H "Content-Type: application/json" \
  -d '{"text":"Hola. Esta es una prueba en espanol.","voice":"ef_dora","speed":1.0,"format":"mp3"}' \
  --output espanol.mp3

# German (Edge TTS)
curl -X POST http://localhost:8880/tts \
  -H "Content-Type: application/json" \
  -d '{"text":"Hallo. Dies ist ein Test auf Deutsch.","voice":"de_f_katja","speed":1.0,"pitch":0,"format":"mp3"}' \
  --output deutsch.mp3

JavaScript / Fetch

async function speak(text, voice = "pt_f_menina", speed = 1.08, pitch = 46) {
  const response = await fetch("http://localhost:8880/tts", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ text, voice, speed, pitch, format: "mp3" }),
  });

  const blob = await response.blob();
  new Audio(URL.createObjectURL(blob)).play();
}

Python / Requests

import requests

BASE = "http://localhost:8880"

resp = requests.post(f"{BASE}/tts", json={
    "text": "Hello! Testing the API with child voice.",
    "voice": "pt_f_menina",
    "speed": 1.08,
    "pitch": 46,
    "format": "mp3",
})

with open("audio.mp3", "wb") as f:
    f.write(resp.content)

PHP / cURL

<?php
$ch = curl_init('http://localhost:8880/tts');
curl_setopt_array($ch, [

    CURLOPT_POST           => true,
    CURLOPT_HTTPHEADER     => ['Content-Type: application/json'],
    CURLOPT_POSTFIELDS     => json_encode([

        'voice'  => 'pt_f_menina',
        'speed'  => 1.08,
        'pitch'  => 46,
        'format' => 'mp3',
    ]),
]);
